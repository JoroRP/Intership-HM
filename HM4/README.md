# School management system CLI Application

This School Management System is a PHP-based console application following OOP principles that allows administrators,
teachers, and students to manage subjects, users, and grades. This system supports user authentication and role-based
access for different functionalities.

## Features

- **Login System**: Authenticate users based on their roles (admin, teacher, student).
- **Admin Menu**: Create or remove subjects, and manage teachers and students.
- **Teacher Menu**: View assigned subjects and grade students.
- **Student Menu**: (Pending implementation) View grades for subjects.

## Usage

You need to start the application from the console by using **php index.php**
