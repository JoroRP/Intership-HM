# Pokemon Memory Game

## Description

The **Pokemon Memory Game** is a browser-based memory game where players match pairs of Pokémon images. The goal is to
find all the matching Pokémon pairs within a time limit. The game features dynamic grid sizing, allowing players to
choose between different grid sizes (2x3 or 4x4). Pokémon images are fetched from
the [Pokémon API](https://pokeapi.co/), making each game different.

## Usage

1. Open the `MemoryGame.html` file in your browser.
2. Choose the grid size (2x3 or 4x4).
3. Click the **Start Game** button.
4. The game will begin with the timer counting down.
5. Match all Pokémon pairs before time runs out!
