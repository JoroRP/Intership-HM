# HM5 - HTML pages with Bootstrap based on a Figma template

This is an HTML training example which uses Vanilla HTML and CSS and Bootstrap libraries.
The 3 pages are made to resemble a template created with Figma.

- The Navbar,Sidebar and Footer which are identical for all pages are moved to separate files and included in each page
  for cleaner code.
- The transaction_history.php has an example of dynamic information pull from a PHP array which gets displayed in a
  table.
